# TicTacToe

A simple TicTacToe game implemented in Java.  
Supports **Player vs Player** and **Player vs AI** modes.

---
## How to Run
1. Clone this repository 
```bash
git clone https://github.com/MichelleVi09/TicTacToe.git
cd TicTacToe
```
2. Compile and run this program:
```bash 
javac src/*.java
java -cp src Main
